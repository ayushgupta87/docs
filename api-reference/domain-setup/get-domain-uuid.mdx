---
title: 'Get Domain UUID'
openapi: 'POST /v1/api/get_domain_uuid/'
---

Retrieve the UUID for a specific domain by providing the domain name and protocol type.

## Example request

<RequestExample>

```bash cURL
curl -X POST https://api.hookpulse.io/v1/api/get_domain_uuid/ \
  -H "x-hookpulse-api-key: {{x-hookpulse-api-key}}" \
  -H "x-brand-uuid: {{x-brand-uuid}}" \
  -H "Content-Type: application/json" \
  -d '{
    "domain_name": "{{domain_name}}",
    "protocol_type": "{{protocol_type}}"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.hookpulse.io/v1/api/get_domain_uuid/', {
  method: 'POST',
  headers: {
    'x-hookpulse-api-key': '{{x-hookpulse-api-key}}',
    'x-brand-uuid': '{{x-brand-uuid}}',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    domain_name: '{{domain_name}}',
    protocol_type: '{{protocol_type}}'
  })
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

url = 'https://api.hookpulse.io/v1/api/get_domain_uuid/'
headers = {
    'x-hookpulse-api-key': '{{x-hookpulse-api-key}}',
    'x-brand-uuid': '{{x-brand-uuid}}',
    'Content-Type': 'application/json'
}
payload = {
    'domain_name': '{{domain_name}}',
    'protocol_type': '{{protocol_type}}'
}

response = requests.post(url, headers=headers, json=payload)
print(response.json())
```

```ruby Ruby
require 'net/http'
require 'json'
require 'uri'

uri = URI('https://api.hookpulse.io/v1/api/get_domain_uuid/')

http.use_ssl = true

request = Net::HTTP::Post.new(uri.path)
request['x-hookpulse-api-key'] = '{{x-hookpulse-api-key}}'
request['x-brand-uuid'] = '{{x-brand-uuid}}'

  domain_name: '{{domain_name}}',
  protocol_type: '{{protocol_type}}'
}.to_json

```

```php PHP
<?php

$url = 'https://api.hookpulse.io/v1/api/get_domain_uuid/';
$data = [
    'domain_name' => '{{domain_name}}',
    'protocol_type' => '{{protocol_type}}'
];

$ch = curl_init($url);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($data));
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'x-hookpulse-api-key: {{x-hookpulse-api-key}}',
    'x-brand-uuid: {{x-brand-uuid}}',
    'Content-Type: application/json'
]);

$response = curl_exec($ch);
curl_close($ch);

echo $response;
?>
```

</RequestExample>

## Request body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `domain_name` | string | Yes | Domain name to look up (e.g., "api.hookpulse.io") |
| `protocol_type` | string | Yes | Protocol type for the domain (e.g., "https", "http") |

## Example response

<ResponseExample>

```json
{
  "success": true,
  "domain_uuid": "{{domain_uuid}}"
}
```

</ResponseExample>

## Response fields

| Field | Type | Description |
|-------|------|-------------|
| `success` | boolean | Indicates if the request was successful |
| `domain_uuid` | string | Unique identifier for the domain |
