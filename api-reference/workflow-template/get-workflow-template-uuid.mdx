---
title: 'Get Workflow Template UUID by Name'
openapi: 'POST /v1/api/get_workflow_template_uuid/'
---

Retrieve the UUID of a workflow template by providing its name.

## Base URL

All API requests should be made to:

```
https://api.hookpulse.io
```

## Example request

<RequestExample>

```bash cURL
curl -X POST https://api.hookpulse.io/v1/api/get_workflow_template_uuid/ \
  -H "x-hookpulse-api-key: {{x-hookpulse-api-key}}" \
  -H "x-brand-uuid: {{x-brand-uuid}}" \
  -H "Content-Type: application/json" \
  -d '{
    "workflow_name": "{{workflow_name}}"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.hookpulse.io/v1/api/get_workflow_template_uuid/', {
  method: 'POST',
  headers: {
    'x-hookpulse-api-key': '{{x-hookpulse-api-key}}',
    'x-brand-uuid': '{{x-brand-uuid}}',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    workflow_name: '{{workflow_name}}'
  })
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

url = 'https://api.hookpulse.io/v1/api/get_workflow_template_uuid/'
headers = {
    'x-hookpulse-api-key': '{{x-hookpulse-api-key}}',
    'x-brand-uuid': '{{x-brand-uuid}}',
    'Content-Type': 'application/json'
}
payload = {
    'workflow_name': '{{workflow_name}}'
}

response = requests.post(url, headers=headers, json=payload)
print(response.json())
```

```ruby Ruby
require 'net/http'
require 'json'
require 'uri'

request['x-hookpulse-api-key'] = '{{x-hookpulse-api-key}}'
```

```php PHP
<?php

$url = 'https://api.hookpulse.io/v1/api/get_workflow_template_uuid/';
$data = [
    'workflow_name' => '{{workflow_name}}'
];

$ch = curl_init($url);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($data));
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'x-hookpulse-api-key: {{x-hookpulse-api-key}}',
    'x-brand-uuid: {{x-brand-uuid}}',
    'Content-Type: application/json'
]);

$response = curl_exec($ch);
curl_close($ch);

echo $response;
?>
```

</RequestExample>

## Request body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `workflow_name` | string | Yes | Name of the workflow template |

## Example response

<ResponseExample>

```json
{
  "success": true,
  "workflow_uuid": "{{workflow_uuid}}"
}
```

</ResponseExample>

## Response fields

| Field | Type | Description |
|-------|------|-------------|
| `success` | boolean | Indicates if the request was successful |
| `workflow_uuid` | string | Unique identifier for the workflow template |
