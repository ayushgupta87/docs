---
title: 'Get Resources'
openapi: 'GET /resources'
---

Retrieve a list of resources from your Hookpulse account. Supports pagination and filtering.

## Example request

<RequestExample>

```bash
curl -X GET https://api.hookpulse.io/v1/api/resources \
  -H "x-hookpulse-api-key: YOUR_API_KEY" \
  -H "x-brand-uuid: YOUR_BRAND_UUID" \
  -H "Content-Type: application/json"
```

</RequestExample>

## Query parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `limit` | integer | Maximum number of results to return (default: 20, max: 100) |
| `offset` | integer | Number of results to skip for pagination (default: 0) |

## Example response

<ResponseExample>

```json
{
  "data": [
    {

    }

```

</ResponseExample>
